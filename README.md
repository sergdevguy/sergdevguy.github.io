# Сайт для хранения всяких штук
- Next.js + Vercel
- MUI (Material UI for React)

## Функционал
- Обычный многостраничный сайт, частично SSR, частично CSR

## Цель
- Попробовать роутинги, SSR, перенос реакт приложений на сайт, посмотреть на многостраничный сайт в действии, изучить структуру файлов реакт проектов.
- Next.js используется для роутинга и изучения структуры файлов многостраничных сайтов
- Оформление с помощью библиотеки MUI, так же, в учебных целях.
- Отслеживаю быстродейсвтие по гугл пейдж спид.

## Выводы
- Next.js требует соблюдение своей структуры проектов в плане директорий и в плане стилей. Эти структуры выглядят хорошо и приняты на вооружение :)
- Как и на других проектах, пока что не хватает более грамотного планирования структур куда.
